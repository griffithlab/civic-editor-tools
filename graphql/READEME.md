To get json graphql Query and Variables objects you can follow these steps:

- Go to the view of interest in CIViCdb.org
- Right Click -> Inspect
- Go to Network tab
- In the list under the Name heading, find entries labelled "graphql" and select those
- Right click on the "query" entry and select "Copy string contents". Paste into a query .json file
- Right click on the "variables" entry and select "Copy object". Paste into a variables .json file
- Replace example IDs in the variables files with the pattern "graphql_query_id1"
